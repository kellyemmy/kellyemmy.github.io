## 🌍 About Me

## 🛠️ Built With
This portfolio was intentionally built without heavy frameworks to demonstrate core frontend fundamentals:
* **HTML5:** Semantic markup and accessibility best practices.
* **CSS3:** Custom CSS variables, modern Grid/Flexbox layouts, and a mobile-first responsive design.
* **Vanilla JavaScript:** Smooth scrolling and dynamic UI interactions.
* **Version Control:** Git & GitHub.
